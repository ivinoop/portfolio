## Portfolio blog built with Hugo. 
---
This site is based off of PaperMod theme, modified by [adityatelange](https://adityatelange.github.io/hugo-PaperMod/) and [arkalim](https://adityatelange.github.io/hugo-PaperMod/).

- Sticky header
- Persistent light & dark modes
- Responsive
- Modifiable navigation menus 
- Hero/Profile section on home page
- Markdown based content management
- Code syntax highlighting with modifiable highlighter themes

---
## Installation

- Clone this repo from [here](https://github.com/ivinoop/portfolio)
- Make sure to have `hugo` command line tools installed
  - [Install hugo](https://gohugo.io/installation/)
- `cd` into the repo and run `hugo server` to run the site locally
